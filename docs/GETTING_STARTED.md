## Using the Stack

ETHRAEON is a constitutional governance runtime — not a chatbot, not a wrapper, not a model. It is infrastructure for AI that must remain human-sovereign.

To use the stack:
- **SaaS/API access**: Request via [ethraeon.ai/contact/](https://ethraeon.ai/contact/)
- **Research/academic**: Cite papers at [papers.ethraeon.ai](https://papers.ethraeon.ai)
- **Enterprise integration**: Brief package at [ethraeon.ai/dataroom/](https://ethraeon.ai/dataroom/)
- **Developer exploration**: All public surfaces at [ethraeon.systems](https://ethraeon.systems)
